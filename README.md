# Arcana Canvas

A canvas-based card interaction system for tarot, playing cards, and custom decks.

## ✨ Overview

Arcana Canvas is a lightweight, interactive card environment built for intuitive, tactile card handling in the browser.

It began as a digital playing card tool and is evolving into a flexible system for working with symbolic decks such as tarot, oracle cards, and beyond.

## 🧩 Features

- Smooth canvas-based card rendering
- Click, draw, and interact with cards naturally
- Designed for real-time card handling (no rigid UI constraints)
- Minimal interface, focus on interaction

## 🔮 Vision

Arcana Canvas is not just a tarot app — it is a foundation for working with symbolic systems.

Planned direction includes:

- Multiple deck support (tarot, playing cards, oracle, runes)
- Custom deck integration
- Spread layouts and history tracking
- Card animations and transitions
- Expandable content system

## 🐉 Future Expansions

- **Dragons of Eldrune** — a custom oracle deck based on Veldara’s discipline system
- Additional themed decks and symbolic systems
